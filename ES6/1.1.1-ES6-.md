#### ECMAScript, JavaScript, NodeJS 区别

``` javascript
///////////////////////////////////////////////////////////////////////////////////////////////////////////////////////


// ECMAScript, JavaScript, NodeJS 区别


//- - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - -//


/**
 * ECMAScript
 * 
 * 简称 ES( 语言标准: 循环，判断，变量，数组等数组类型 )
 */


/**
 * JavaScript
 * 
 * 运行在浏览器端的语言，该语言使用 ES 标准
 * 
 * ES + webAPI = JavaScript
 */


/**
 * NodeJS
 * 
 * 运行在服务器端的语音，该语言使用 ES 标准
 * 
 * ES + nodeAPI = JavaScript    
 */


///////////////////////////////////////////////////////////////////////////////////////////////////////////////////////
```
