    * find && mdfind
    
        * find: 用于具体检索
        
            * find pathName -options [actions] 'fileName'
            
                * find ~ -name '*fileName.expanded*' => 全名检索( 当前目录，包含子目录 )
            
                * find ~ -name '*ExpandedName' => 后缀检索( 当前目录，不包含子目录 )

                * find ~ -name 'fileName*' => 前缀检索( 当前目录，不包含子目录 )
                 
                * 
                
        * mdfind: 快速全盘检索
             
            * mdfind fileName == mdfind . fileName => 快速检索文件或文件夹
            
                * mdfind FileName 
            
                * mdfind -onlyin pathName fileName
                
                * 
             
        * 必须添加，否则会出现很多类型文件
         
            * -name => 根据文件名查找，区分大小写
                   
            * -iname => 根据文件名查找，不区分大小写

* ### cursor cmd

    * control + l => $
    
        * move line before

        *
    
    * control + e => ^
    
        * move line after

        *
    
    * control + u => dd
    
        * delete a line

        *
    
    * control + k => d$
    
        * delete cursor after content

        *
    
    * option + ( <- | -> ) => w | b
    
        * jump word before or after

        *
