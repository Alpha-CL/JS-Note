# Document Object Mode

DOM 定义了表示和修改文档所需的方法

DOM对象 及为宿主对象，由浏览器厂商定义，用来操作 html 和 xml 功能的一类对象的合集

DOM 也称为 HTML 以及 XML 的标准编程接口
