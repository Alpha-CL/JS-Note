    * .methods(); => 一次添加一个方法
    
        *

        * syntax: 
        
            * $(jqObj).bind({
            
                * method1.function(){
                
                    * ... statements  ...
                
                * },
                
                * method2.function(){
                
                    * ... statements ...
                                
                * }
            
            * });
    
            *

            * $(jqObj).delegate(selector, types, date, fn);   

            * selector: 最终事件执行元素
            
            * types: 事件类型
