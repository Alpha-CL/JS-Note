* ### 搭建 web 服务器

    * 服务器( 提供服务 ): 安装特定软件的公共计算机，用于提供特定服务
    
        * 按照服务类型不同，划分为:
        
            * Web 服务器
            
            * 数据库服务器
            
            * 文件服务器
            
            *  
            
    * 客户端( 使用服务 ): 在一次服务过程中使用这个服务器的设备( 网络端点 )
    
        * 目前常见的客户端 => 浏览器
        
        * 
        
* ### Web 服务器软件( 不是捆绑搭配关系 )

    * NgInx --> 反向代理
    
    * Apache --> PHP
    
    * IIS --> ASP.NET
    
    * Tomcat --> Java
