# HTML

#### HTML Structure

``` javascript


// 因为 html 的更新迭代，和个大浏览器对 html 版本的支持
// 标记出 html 的版本，有利于浏览器执行 html 时选择对应 html 版本的规范解析
// 若不写 html version，则会启动 html 混杂模式

<!DOCTYPE html>                                 // html version: 为了向前兼容

/**
 * lang 属性是全局属性，表示该元素内容使用的文字是哪一种自然语言书写而成的
 * 
 * 'cmn-hans': 标准中文简体
 */

<html lang="en">                                // root tag start

<head>                                          // head tag start       // head 中的元素都不显示到浏览器页面中

    /**
     * 计算机中使用  0 1 
     * 
     * 低电压(0 ~ 2)V => 0, 高电压(0 ~ 5)V => 1
     */

    <meta charset="UTF-8">      // Character Encoding           // 文档元数据，附加信息
    
    <title>Title</title>        // Page title on browser tab
    
</head>                                         // head tag end

<body>                                          // body tag start

    // Page display contain

</body>                                         // body tag end

</html>                                         // root tag end


```


#### Element

``` javascript


/**
 * Element
 * 
 * @tagPart: eleName: <begin tag>                   // 大多数标签都是成对出现
 * 
 * @tagPart: eleAtribute 
 *                             
 *           eleAttributeName = "eleAttributeValue" // 大多数属性都是成对出现
 * 
 * @tagPart: eleContain                             // 大多数标签内容是展示给用户看的内容
 * 
 * @tagPart: eleName: </end tag>                       
 * 
 */

<'eleName: begin tag' + 'eleAtribute'> 'eleContain' </'eleName: end tag'>

eg: <div></div>
    <p><p>
```

#### tag Semantic

``` javascript


/**
 * 为后期维护及写作文，应使用以具有语义话的 html 搭建框架
 * 
 * 选择元素标签，源于内容的含义
 * 
 * 所有元素与页面展示效果无关
 * 
 * 
 * => 1. 为了搜索引擎优化( SEO )    // 每隔一段时间，搜素引擎会从整个互联网中抓取页面源代码
 * 
 *    2. 为了让浏览器理解网页        // 阅读模式，语音模式
 * 
 */

// example

<a></a>             // 超链接

<p></p>             // 文字段落

<h1></h1>           // 段落标题


```

#### 元素包含关系

> 以前: 块级元素可以包含行级元素，行级元素不可以包含块级元素，a 元素除外

``` javascript
///////////////////////////////////////////////////////////////////////////////////////////////////////////////////////


1) a 元素中机会可以包含任何元素

2) 部分元素只能包含固定的子元素( eg: ul, li, map ...)

3) 标题元素和段落元素不能相互嵌套，并且不能包含容器元素


///////////////////////////////////////////////////////////////////////////////////////////////////////////////////////
```
