# HTML

#### HTML Structure

``` javascript


// 因为 html 的更新迭代，和个大浏览器对 html 版本的支持
// 标记出 html 的版本，有利于浏览器执行 html 时选择对应 html 版本的规范解析

<!DOCTYPE html>                                 // html version: 为了向前兼容

<html lang="en">                                // root tag start

<head>                                          // head tag start

    <meta charset="UTF-8">      // Character Encoding
    
    <title>Title</title>        // Page title on browser tab
    
</head>                                         // head tag end

<body>                                          // body tag start

    // Page display contain

</body>                                         // body tag end

</html>                                         // root tag end



```


#### Element

``` javascript


/**
 * Element
 * 
 * @tagPart: eleName: begin tag                     // 大多数标签都是成对出现
 * 
 * @tagPart: eleAtribute 
 *                             
 *           eleAttributeName = "eleAttributeValue" // 大多数属性都是成对出现
 * 
 * @tagPart: eleContain                             // 大多数标签内容是展示给用户看的内容
 * 
 * @tagPart: eleName: end tag                       
 * 
 */

<'eleName: begin tag' + 'eleAtribute'> 'eleContain' </'eleName: end tag'>

eg: <div></div>
    <p><p>


```
