# HTML5 Attribute

#### HTML5 new add type of input

``` javascript
///////////////////////////////////////////////////////////////////////////////////////////////////////////////////////


<!-- HTML5 Calendar 类( 兼容性都不好，多数仅 chrome 支持 ) -->

<input type="date">                             <!-- 日期选择器: chrome 支持，safari & IE 不支持-->

<input type="time">                             <!-- 时间选择器 -->

<input type="week">                             <!-- 周选择器 -->

<input type="datetime-local">                   <!-- 年月日时分秒 选择器-->

<input type="number">                           <!-- 数字输入器( 仅可输入数字 ): chrome 支持，safari & IE 不支持 -->

<input type="email">                            <!-- 邮箱输入框: chrome & 火狐 支持，safari & IE 不支持 -->

<input type="color">                            <!-- 颜色 选择器: chrome 支持，safari & IE 不支持 -->

<input type="range">                            <!-- 滑动条: 可设置最大/小值，获取用于选择的值 -->
                                                <!-- chrome safari 支持 火狐 IE 不支持 -->

<input type="search" name="search">             <!-- 保留搜索历史供后续检索时提示 -->
                                                <!-- chrome 支持 safari部分支持 IE不支持 -->

<input type="url">                              <!-- 仅可输入完整的网址 -->
                                                <!-- chrome 火狐 支持 safari IE 不支持 -->


///////////////////////////////////////////////////////////////////////////////////////////////////////////////////////
```


#### content edit able

> 没有兼容问额

``` javascript
///////////////////////////////////////////////////////////////////////////////////////////////////////////////////////


/**
 * contenteditable
 * 
 * 可继承，无兼容，但 delete 可以删除内部标签元素
 */

<div contenteditable="ture"></div>

<div contenteditable="true">                        <!-- 没有兼容问题 -->
    Panda
    <br>
    <span>Monkey</span>                             <!-- 可继承 -->
</div>

<br>

<!--example-->                                           <!-- 可以删除内部标签 -->

<div contenteditable="true">
    <span contenteditable="false">姓名: </span>alpha      <!-- br 会被删除 -->
    <br>
    <span contenteditable="false">姓别: </span>male
</div>


///////////////////////////////////////////////////////////////////////////////////////////////////////////////////////
```
